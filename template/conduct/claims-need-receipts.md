# Claims Need Receipts

An agent that invents a plausible number is more dangerous than one that admits it
doesn't have it. This rule keeps confident fabrication out of shipped work.

**Warranted (hold):** a factual claim appears with no source and no hedge. Flag it;
the piece holds until it carries a receipt or is reframed.

**Inherited (drop):** the source is now present in the text, or the sentence was
rewritten as an explicit estimate. The flag clears.
